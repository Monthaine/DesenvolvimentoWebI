Revisão Prática de Desenvolvimento WEB I + CSS

Seção 01- Conceitos e Fundamentos

  • Linguagem de Marcação:Linguagem de marcação é o sistema que usa marcações para criar e estruturar a página web.
  • HTML:é a linguagem de marcação (não é de programação!) usada para criar e estruturar a página web.
  • TAG:é o elemento do HTML que define a função de um conteúdo (p, h1, etc.)
  • Atributo:é a propriedade adicionada à uma tag para modificar ou dar detalhe (ex: href)
  • Valor:é a informação atribuida a um atributo que define a sua configuração.

Seção 02- Exemplo de Estrutura HTML

Os documentos html possuem uma estrutura básica, a qual geralmente compõe a maioria das páginas html. Veja o exemplo abaixo.

			 
				<!DOCTYPE html>
				<html lang="pt-br">
					<head>
						<title>Aqui vai o título</title>
						<meta charset="utf-8">
					</head>
					<body>
					</body>
				</html>

Podemos observar que o código html inicia-se com a abertura da tag "html" e sua estrutura básica constitui de cabeça (head) e corpo (body). Para indicarmos que é o HTML 5, indicamos por !DOCTYPE html. A página será interpretada como lingua brasileira, uma vez que o charset está em "pt-br". Para que seja interpretado todos os caracteres da língua portuguesa, indicamos o charset em utf-8. Titles indicam informações da aba, head, informações no cabeçalho da página e body indica toda a estrutura visível da página web.
